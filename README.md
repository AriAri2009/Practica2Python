# Práctica 2: Gestión de Entornos y Control de Versiones
#Desarrollado por: Arianna Cedeño

## Descripción del Proyecto
Este proyecto consiste en una aplicación de consola desarrollada en Python que demuestra el uso de buenas prácticas de desarrollo. Se ha implementado un sistema de gestión de dependencias y entornos virtuales para asegurar la portabilidad del código, además de integrar una librería externa para la comunicación con APIs.

## Especificaciones Técnicas
* **Versión de Python utilizada:** 3.14.3
* **Gestor de Dependencias:** Poetry
* **Librerías externas:** `requests` (para la gestión de peticiones HTTP).

## Instrucciones de Instalación
Para configurar el entorno de desarrollo y descargar las dependencias necesarias, asegúrese de tener instalado Poetry y ejecute el siguiente comando en la raíz del proyecto:

```bash
poetry install
